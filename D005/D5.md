# 퀴즈
1. 문자열 앞뒤 공백 제거 및 소문자 통일
정답: (b) .str.strip().str.lower()

설명: .str.strip()이 글자 앞뒤에 붙은 불필요한 빈칸(공백)을 깎아내고, .str.lower()가 알파벳 대문자를 모두 소문자로 변환해 줍니다.

2. 제각각이지만 규칙이 있는 패턴을 뽑는 도구
정답: 정규 표현식 (Regular Expression, 줄여서 RegEx)

설명: 이메일 형식, 전화번호, 로그 안의 IP 주소처럼 위치는 제각각이어도 "숫자-글자-숫자" 같은 특정한 규칙(패턴)을 가지고 있는 텍스트를 정확하게 찾아내고 추출해 주는 마법 같은 텍스트 검색 도구입니다. (판다스에서는 주로 .str.extract()와 함께 사용합니다.)

3. 날짜 오류값을 NaT(결측치)로 처리하는 옵션
정답: errors='coerce'

설명: 원래 날짜 변환 중에 글자나 깨진 데이터 같은 오류를 만나면 에러를 뿜으며 실행이 멈추지만, 이 옵션을 주면 *"에러 내지 말고 그냥 모르는 날짜(NaT, Not a Time)로 비워두고 넘어가!"*라고 강제 지정해 줍니다.

코드 예시: pd.to_datetime(df['date_column'], errors='coerce')

4. 초 단위 데이터를 "하루 단위 합계"로 묶는 메서드와 빈도 문자
정답: .resample('D').sum() (또는 .groupby())

설명: 시간 간격을 다시 조절하는 resample() 메서드에 하루를 뜻하는 빈도 문자 'D'(Day)를 넣고, 값들을 다 더해주는 .sum()을 붙여주면 초 단위 데이터가 깔끔하게 일별 합계로 뭉쳐집니다.

# 코드 퀴즈 — 모범 답안
# 1) 정제(strip+lower) → 2) @ 뒤 도메인 추출 → 3) 조건 집계
clean = customers["email"].str.strip().str.lower()

gmail_count = (domain == "gmail.com").sum()
