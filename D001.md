# 옷장패션 데이터 정제 보고서

## 1. 데이터 개요
- 행/열: 1,500 × 7
- 주요 컬럼: order_id, customer_age, category, channel, price, quantity, amount, return_amount

## 2. 진단 결과
- 결측: amount 3.40%, price 0.33%, ...
- 이상치(IQR): customer_age(999·0, 18건), quantity(200, 1건), amount(50,000,000, 145건)
- 의심되는 결측 유형: amount는 app 채널에 쏠림 → MAR

## 3. 처리 결정과 근거
| 이슈 | 결정 | 근거 | 한계 |
| coustomer_age | nan 처리 후 중앙값 대체 | 물리적 불가능 값 | 특정 연령대로 쏠릴수있음 |
| amount 결측 | 채널별 중앙값 대체 | MAR 가설 부합 | 개별데이터 정확성은 떨어짐|
| amount 이상치| 유지 + 플래그 | 실제 매출 데이터일수도 | 그대로두면 평균이나 중앙값에 영향 줌 |

## 4. 처리 후 검증
- 결측 0건(설계상 NaN 유지가 필요한 컬럼 제외)
